erldocs.com

Welcome, erldocs is an alternative to the official erlang documentation. The code to generate the documentation can all be found on github, and if you have any bugs to report, please chuck them in issues.

If you are looking for more erlang resources, please checkout erlangotp.com, and if you are feeling nice you can flattr me on the right.
